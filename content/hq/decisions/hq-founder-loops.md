Panel session (Jobs, Norman, Grove, Drucker, Cook lenses) on reorganising
HQ as a sole founder-operator tool for at least two years.

**Finding.** 28 routes, 11 hubs, and a nav of six hardcoded noun links that
matched neither. The home page already answers "what now" (proof spine,
metrics, operator todos); the debt was wayfinding between rooms.

**Decision.** Everything a sole founder does is one of four verbs:
- **Sell** — CRM, marketing engine
- **Make** — Design Rooms, Asset Command, asset library
- **Tell** — reporting, shareholders, data room
- **Run** — vault, org, blueprint

The nav is now those verbs (+ board). The hub board groups by loop.
Design Rooms joins the hub map. Deep rooms stay reachable via the board
and the command palette; nothing was deleted.

**Rejected.** Merging Assets into Asset Command (reference vs gate is a
real distinction); a sidebar (the 44px bar and palette suffice for one
operator); renaming routes (bookmarks and muscle memory survive).
