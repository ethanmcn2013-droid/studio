## Format

**Trim size:** A5 (148 × 210mm) portrait
**Page count:** 16pp (cover + 14 interior pages + back cover)
**Binding:** Saddle stitch (recommended for ≤20pp, allows lay-flat reading)

**Why A5:** Intimate scale. Sits naturally in the hand. Can be read on a train or at a table. The certificate is A4; the handbook being A5 creates a deliberate scale relationship — the certificate is for framing, the handbook is for reading. They complement rather than compete.

## Color Application

All interior pages: Ink (#1C1917) on Ivory (#F9F7F2).
No tinted panels, no grey fills, no color except black ink on warm white paper.
All gradation of visual hierarchy is achieved through type size, weight, and whitespace — not color.

Chapter numbers: Graphite (#5A5450) — the only lighter value used in body pages.
Dividing rules: Stone (#C4BBB0) hairline (0.5pt).

## Print Specification

### Covers
- **Stock:** Conqueror Laid Natural, 300gsm
- **Color:** Black only (Ink #1C1917; print as C20 M15 Y10 K90 for depth)
- **Finish:** Blind emboss — the wordmark "signal studio ·" on the front cover is embossed into the stock, no ink. This is the only finish treatment on the cover.
- **Note:** No foil. No varnish. No UV.

### Interior
- **Stock:** Conqueror Laid Natural, 120gsm (or Fedrigoni Arcoprint Natural EW, 120gsm)
- **Color:** Black only
- **Finish:** None — uncoated, untreated

### Binding
- **Recommended:** Saddle stitch with matching ivory thread (or standard wire)
- **Alternative for keepsake feel:** Hand-sewn Coptic stitch binding — opens flat, beautiful, approximately €20–30 extra per unit for a bookbinder. Worth considering for Mam's copy.

### Dimensions after binding
- Trim: 148 × 210mm (A5)
- Spine: approximately 3mm for 16pp at 120gsm interior + 300gsm cover

### Print run
- 5 copies (one per recipient)
- Recommend 3 extra: one archive, one author's copy, one reprint buffer
- Total print run: 8 units

### Supplier notes
- Most premium digital print shops in Ireland (e.g., Snap, BNL) can handle this run on Conqueror stock
- For blind emboss, a specialist print finisher or letterpress shop is required — allow 3–4 weeks
- Print deadline: approximately August 1 to allow binding, packaging assembly, and delivery

## Pagination and Imposition

For a 16pp saddle-stitch booklet, the imposition is:

| Sheet | Side A         | Side B         |
|-------|----------------|----------------|
| 1     | p.16 / p.1     | p.2 / p.15     |
| 2     | p.14 / p.3     | p.4 / p.13     |
| 3     | p.12 / p.5     | p.6 / p.11     |
| 4     | p.10 / p.7     | p.8 / p.9      |

4 sheets, 8 sides, saddle-stitch through center fold.

This is standard for any print supplier — provide a PDF with sequential pages (1–16) and request saddle-stitch imposition.

## Typography PDF Setup

When creating the final print PDF:
- Export from InDesign, Affinity Publisher, or equivalent
- Embed all fonts
- Color space: CMYK (not RGB)
- Resolution: 300dpi minimum for any images
- Bleed: 3mm all sides (for cover only — interior pages do not require bleed)
- Marks: crop marks + bleed marks
- PDF/X-4 format preferred by most commercial printers
