# SEO And Template Strategy

Templates are acquisition surfaces, not only setup shortcuts.

Early template pages should:

- solve one concrete problem
- show the workspace artefact clearly
- explain what the user gets
- make duplication obvious
- keep the broader Signal Studio loop visible

Priority template pages:

- wedding planning workspace
- venue walkthrough checklist
- supplier comparison board
- 12-month wedding roadmap
- final-week wedding checklist
- student group project workspace
- society event planning workspace
- freelancer client delivery workspace
