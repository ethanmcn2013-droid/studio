# Review Checklist

Before any asset is approved, check:

## Brand

- Is it plain English?
- Is it useful?
- Is it calm?
- Does it avoid jargon?
- Does it avoid hype?

## Strategy

- Which segment is it for?
- Which campaign does it support?
- Which product or ecosystem loop does it make clearer?
- Which metric could it move?

## Growth

- Can it become more than one asset?
- Does it support a shareable output, template, invite, pilot, or demo?
- Is there a clear next action?

## Trust

- Are claims accurate?
- Are testimonials real?
- Is outreach respectful?
- Is there any privacy or consent concern?
